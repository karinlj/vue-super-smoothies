# Super-smoothies Vue-app

## Description

Small vue app with login and CRUD with Firebase as backend.

## Prerequisites

NodeJS

## Installation

To clone

`git clone https://github.com/karinlj/vue-super-smoothies.git`

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```
